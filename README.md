Sales Data Validation & KPI Analysis
Project Overview

This project demonstrates how a Business Analyst performs data validation, KPI analysis, and reporting support using SQL and Python.

The objective was to validate sales transaction data, identify inconsistencies, and calculate key business metrics to support dashboard reporting.

Business Problem

The organization needed accurate sales performance metrics to support executive reporting.
However, raw transaction data contained inconsistencies, missing values, and duplicate entries.

Approach

Validated sales records using SQL queries

Cleaned and transformed data using Python (Pandas)

Calculated KPIs such as Revenue, Average Order Value, and Monthly Growth

Prepared structured dataset for dashboard reporting

Tools Used

SQL

Python (Pandas)

Excel

Power BI (conceptual reporting layer)

Key KPIs Calculated

Total Revenue

Monthly Revenue Growth

Average Order Value

Top Performing Regions

Outcome

Improved reporting accuracy and ensured reliable KPI dashboards for business stakeholders.
